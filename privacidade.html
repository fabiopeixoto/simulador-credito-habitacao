Política de Privacidade

Última actualização: Junho de 2026

1. Quem somos

O Simulador Crédito Habitação (simhabitacao.pt) é uma ferramenta gratuita de apoio à decisão que permite comparar condições de crédito habitação em Portugal. Não somos um banco, intermediário de crédito, nem prestador de serviços financeiros.

⚠️ Os valores apresentados são estimativas meramente indicativas com base em dados públicos. Não constituem aconselhamento financeiro nem proposta de crédito.

2. Dados que recolhemos

A nossa recolha de dados é mínima e limita-se ao necessário para o funcionamento da plataforma:

  • Registos de acesso (logs): endereço IP, data/hora, página visitada e browser. Mantidos no servidor por 30 dias para fins de diagnóstico e segurança.
  • Estatísticas agregadas de visitas: número de visitantes por dia e localização aproximada por cidade. A geolocalização é determinada localmente nos nossos servidores com base numa base de dados MaxMind GeoLite2 — o teu endereço IP nunca sai dos nossos servidores para este fim; apenas a cidade resultante é armazenada. Os dados de visitas são agrupados e anonimizados — não identificamos utilizadores individuais.
  • Comentários (opcional): se publicares um comentário, guardamos o nome/alcunha que indicares, o texto e a data/hora. Opcionalmente, também são guardados o banco seleccionado, a prestação simulada e a prestação real, caso os preenchas. Não pedimos email nem criação de conta. Podes usar um nome fictício. Qualquer utilizador pode reportar um comentário inadequado ao administrador.
  • Dados da simulação: os parâmetros que introduzes no simulador (capital, prazo, rendimento, etc.) são processados exclusivamente no teu browser e/ou enviados para os nossos servidores apenas para efectuar o cálculo — não são armazenados de forma permanente.

3. Cookies e armazenamento local

Não utilizamos cookies de rastreamento nem cookies de terceiros para publicidade comportamental.

  • localStorage: guardamos as preferências da simulação (valores introduzidos, histórico local) exclusivamente no teu dispositivo, para que não tenhas de os voltar a introduzir. Estes dados nunca saem do teu browser.
  • Service Worker: utilizado para pré-carregar os ficheiros estáticos da aplicação (cache de desempenho). Não intercepta pedidos à API nem guarda dados pessoais.

4. Finalidade e base legal do tratamento

  • Prestação do serviço: processar os parâmetros da simulação e devolver resultados. Base legal: execução do serviço solicitado (art. 6.º, n.º 1, al. b) do RGPD).
  • Segurança e estabilidade: logs de acesso para detectar anomalias e abusos. Base legal: interesse legítimo (art. 6.º, n.º 1, al. f) do RGPD).
  • Melhoria do serviço: estatísticas agregadas de visitas. Base legal: interesse legítimo.
  • Comentários: publicação voluntária de opiniões. Base legal: consentimento (ao submeter o formulário).

5. Partilha de dados com terceiros

Não vendemos nem cedemos dados pessoais a terceiros. Os únicos terceiros envolvidos são:

  • Banco Central Europeu (BCE): consultamos a API pública do BCE para obter valores Euribor actualizados. Nenhum dado teu é transmitido ao BCE.
  • Fornecedor de alojamento: o servidor está alojado numa infraestrutura cloud. Os logs de acesso podem estar sujeitos às políticas do fornecedor.
A fonte Inter e os logótipos dos bancos são servidos directamente pelos nossos servidores — o teu endereço IP nunca é transmitido ao Google ou a outros terceiros para carregamento de recursos visuais.

6. Retenção de dados

  • Logs de acesso: 30 dias.
  • Estatísticas agregadas de visitas: conservadas indefinidamente (são anónimas).
  • Comentários: conservados enquanto o serviço estiver activo. Podes solicitar a remoção do teu comentário (ver secção 7).

7. Os teus direitos (RGPD)

Ao abrigo do Regulamento Geral sobre a Protecção de Dados (RGPD — Regulamento UE 2016/679), tens os seguintes direitos:

  • Acesso: saber que dados temos sobre ti.
  • Rectificação: corrigir dados incorrectos.
  • Apagamento: solicitar a eliminação dos teus dados ("direito ao esquecimento").
  • Limitação do tratamento: restringir o uso dos teus dados em determinadas circunstâncias.
  • Portabilidade: receber os teus dados num formato estruturado.
  • Oposição: opor-te ao tratamento baseado em interesse legítimo.

Para exercer estes direitos, contacta-nos por email: geral@simhabitacao.pt. Responderemos no prazo de 30 dias.

Tens também o direito de apresentar reclamação à CNPD (Comissão Nacional de Protecção de Dados): www.cnpd.pt.

8. Segurança

Adoptamos medidas técnicas adequadas para proteger os dados contra acesso não autorizado, nomeadamente:

  • Cabeçalhos de segurança HTTP (CSP, X-Frame-Options, X-Content-Type-Options, etc.)
  • Ligação HTTPS
  • Validação de entradas e protecção contra injecção

9. Alterações a esta política

Podemos actualizar esta política periodicamente. A data de última actualização está indicada no topo desta página. Em caso de alterações significativas, publicaremos um aviso na página principal.
